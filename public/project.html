OUR PROJECTS

Project 1: To do list app

This To-Do List App is a simple yet powerful task management tool developed using HTML, CSS and Javascript. It allows users to keep track of their daily activities by adding, editing, deleting, and marking tasks as complete. The app provides a clean, responsive user interface that works smoothly across different devices.

Project 2: OI,NAO project

OI, NAO method is a way to extract information from a question regarding logic gate statements. This was our first time in app development and managed to create a simple game about logic gates. In this project, I (Ivor) was the lead developer and through our cooperation, we managed to achieve Gold Medal in PIITRAM 2023.

Project 3: Stopwatch project

A sleek and functional stopwatch built with HTML, CSS, and JavaScript. This app lets users start, stop, and reset the timer with smooth performance and a clean, responsive design — perfect for time tracking or productivity sessions.
